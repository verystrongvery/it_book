배시 설정 파일
- 셸에 설정한 변수나, alias는 셸이 종료되면 전부 없어집니다.
- 값을 유지하려면 배시 설정 파일을 수정해야 합니다.
/etc/profile
- 시스템 전체에 적용되는 배시 설정 파일 입니다,
~/.profile
- 로그인할 때만 사용되는 사용자 별 배시 설정 파일 입니다.
- ~/.profile 에 ~/.bashrc를 실행시키는 코드가 포함되어 있습니다.
~/.bashrc
- 항상 적용되는 사용자 별 배시 설정 파일 입니다.
- 로그인을 할 때 또는 source ~/.bashrc 명령어를 실행 할때 설정이 반영 됩니다.
